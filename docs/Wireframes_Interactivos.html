ATAJOS DE TECLADO GLOBALES:
  Ctrl+F ──────→ Foco en campo de búsqueda
  Ctrl+N ──────→ Abre diálogo "Add Report"
  Ctrl+E ──────→ Inicia exportación a CSV/Excel/PDF
  Ctrl+O ──────→ Ir a "Overview"
  Ctrl+R ──────→ Ir a "Detailed Reports"
  Ctrl+P ──────→ Ir a "Performance"
  Tab ─────────→ Navegar entre elementos
  Shift+Tab ───→ Retroceder en navegación

Flujos Clave Identificados:

  • Flujo 1 - Exploración: Usuario abre app → ve Overview → navega a Reports → Performance
  • Flujo 2 - Búsqueda: Usuario filtra por cliente → busca specific report → click View para detalles
  • Flujo 3 - Creación: Ctrl+N → llena formulario → Guarder → vuelve a Overview con tabla actualizada
  • Flujo 4 - Exportación: Usuario selecciona rango → Ctrl+E → elige formato → descarga archivo
  • Flujo 5 - Análisis: Usuario va a Performance → selecciona período → visualiza gráficas → identifica tendencias

♿ Accesibilidad (WCAG 2.1 - AA)

✅ Implementado

  • Navegación por Teclado: Tab → elementos interactivos
  • Atajos Mnemotécnicos: Ctrl+F (Find), Ctrl+N (New), Ctrl+E (Export)
  • Foco Visible: Border naranja (#f39c12) en elementos
  • Colores Contrastados: WCAG AA 4.5:1 ratio
  • Labels Claros: Todos los botones y campos descritos
  • Indicadores no-solo-color: ✓ ✗ + colores

📋 Por Validar

  • Zoom 200%: Verificar que UI no se rompe
  • Lector de Pantalla: Testear con JAWS/NVDA
  • Modo Oscuro: Considerar tema alterno
  • Tamaño Fuente: Permitir aumentar del lado usuario
  • Animaciones: Respetar prefers-reduced-motion
  • Navegación Completa: Sin ratón (teclado 100%)
🎯 Objetivo: Asegurar que analistas financieros con discapacidades visuales o motoras puedan operar el dashboard sin frustración usando solo teclado.
